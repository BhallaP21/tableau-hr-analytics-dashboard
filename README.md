# HR Analytics Dashboard | Tableau

An interactive Tableau HR Analytics project designed to provide workforce insights, demographic analysis, employee performance tracking, and salary analysis through executive-level dashboards.

---

## Project Overview

This project consists of two fully interactive dashboards:

1. HR Summary Dashboard
2. Employee Details Dashboard

The dashboards help HR managers and business leaders analyze:

- Workforce composition
- Hiring and termination trends
- Employee demographics
- Salary distribution
- Department-wise employee analysis
- Employee-level detailed records

---

## Business Problem

HR teams often struggle to consolidate workforce data for strategic decision-making.

This project was developed to help HR managers:

- Monitor workforce trends
- Analyze employee demographics
- Understand salary patterns
- Evaluate workforce distribution
- Access detailed employee-level insights
- Improve HR reporting efficiency

---

## Dashboard Features

### 1. HR Summary Dashboard

#### Overview Section
- Total Hired Employees
- Active Employees
- Terminated Employees
- Hiring & termination trends over time
- Department-wise employee distribution
- HQ vs Branch employee comparison
- Geographic employee distribution

#### Demographics Section
- Gender ratio analysis
- Employee distribution by age groups
- Education level analysis
- Education vs performance correlation

#### Income Analysis Section
- Salary comparison by education level & gender
- Age vs salary correlation by department

---

### 2. Employee Details Dashboard

#### Employee-Level Analysis
- Employee ID
- Name
- Department
- Position
- Gender
- Age
- Education
- Salary
- Employment status
- Length of employment

#### Interactive Features
- Column-level filtering
- Dynamic employee search
- Drill-down analysis
